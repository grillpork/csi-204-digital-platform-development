## 1. ข้อมูลเชิงเทคนิคระบบ (System Overview)

สถาปัตยกรรมและการทำงานเบื้องหลัง (Under the Hood) ของแพลตฟอร์ม The Shirtsy ครอบคลุมตั้งแต่ Tech Stack สถาปัตยกรรมระบบคลาวด์ การจัดการฐานข้อมูล ไปจนถึงโปรโตคอลความปลอดภัย

### 1.1 สถาปัตยกรรมระบบ (Tech Stack)
* **Frontend (Client-side):** 
  * **Framework:** Next.js (App Router) สำหรับทำ Server-Side Rendering (SSR) เพื่อประสิทธิภาพด้าน SEO และความเร็วในการโหลดหน้าเว็บ
  * **UI Library:** React.js ร่วมกับ Tailwind CSS สำหรับการออกแบบอินเตอร์เฟสที่ตอบสนองอย่างรวดเร็ว (Responsive) และทันสมัย
  * **State Management:** Zustand สำหรับจัดการ Global State ของผู้ใช้และตะกร้าสินค้า
  * **Interactive Design Tool:** ใช้ React Easy Crop และ HTML Canvas ในหน้าระบบ Custom Design เพื่อให้ผู้ใช้ออกแบบ ปรับแต่งภาพสกรีนบนเบราว์เซอร์ได้อย่างไหลลื่น
* **Backend (Server-side):**
  * **API Architecture:** สร้าง RESTful API ผ่าน Next.js Route Handlers (Edge & Node.js Runtimes)
  * **Authentication:** ระบบความปลอดภัยโดยใช้ Custom JWT Auth เก็บใน Cookie (HttpOnly) เพื่อความปลอดภัยสูงสุด
  * **Deployment:** โฮสต์และจัดส่งเนื้อหาผ่าน Vercel และ Global CDN

### 1.2 โครงสร้างข้อมูล (Database & Storage)
* **Database Engine:** PostgreSQL โฮสต์อยู่บน Neon Serverless Postgres ที่สเกลอัตโนมัติเมื่อปริมาณทราฟฟิกพุ่งสูงขึ้น
* **ORM Layer:** ใช้ Prisma ORM ในการจัดการ Schema และคิวรีข้อมูล
* **Media Storage:** อาร์ตเวิร์คและรูปโปรไฟล์จะถูกแยกจัดเก็บไว้บน Cloud Object Storage (เช่น AWS S3 / Cloudinary) เพื่อลดภาระเซิร์ฟเวอร์หลัก โดยระบบจะแปลงภาพเป็นไฟล์ WebP อัตโนมัติเพื่อประหยัดแบนด์วิดท์

### 1.3 ระบบสิทธิ์และการจัดการผู้ใช้ (User Roles)
ระบบแบ่งระดับผู้ใช้งาน (RBAC - Role Based Access Control) ออกเป็น 3 ระดับหลัก:
1. **Customer (ลูกค้าทั่วไป):** ค้นหาและซื้อสินค้า, ออกแบบเสื้อ Custom, จัดการตะกร้า/ที่อยู่, และดูประวัติออเดอร์ของตนเอง
2. **Creator (นักออกแบบ):** ได้รับสิทธิ์เหมือนลูกค้าทั้งหมด พร้อมเปิดใช้ Creator Dashboard เพื่อจัดการหน้าร้านค้า, อัปโหลดลายใหม่, และกำหนดส่วนแบ่งรายได้ (Markup)
3. **Administrator (ผู้ดูแลระบบ):** เข้าถึง Admin Panel เพื่อตรวจสอบสถิติระบบ, จัดการออเดอร์, อนุมัติร้านค้าผู้สร้าง, และตรวจสอบการเงินของแพลตฟอร์ม

### 1.4 ระบบการเงินและคำสั่งซื้อ (Transactions)
* **Stateful Cart:** ระบบตะกร้าสินค้าถูกจัดการร่วมกันระหว่าง Local Storage (สำหรับผู้ใช้ทั่วไป) และจะ Sync ข้อมูลเข้าฐานข้อมูลทันทีเมื่อผู้ใช้งานเข้าสู่ระบบ
* **Real-time Webhook:** เชื่อมต่อกับผู้ให้บริการ Payment Gateway เมื่อสแกนจ่าย QR Code หรือจ่ายผ่านบัตรสำเร็จ ระบบจะรับ Webhook เพื่อเปลี่ยนสถานะเป็น "ชำระเงินสำเร็จ" ทันที
* **Revenue Splitting:** บันทึกแยกสองฝั่งอย่างชัดเจน ได้แก่ "ยอดรวมสำหรับบริษัท" และ "ส่วนแบ่ง Markup สำหรับ Creator" โดยส่วนแบ่งจะโอนเข้ากระเป๋าเงินของ Creator ทันทีเมื่อออเดอร์จัดส่งสำเร็จ

### 1.5 มาตรฐานความปลอดภัย (Security)
> [!IMPORTANT]
> **การเข้ารหัสและสิทธิบัตรข้อมูล:**
> * **Password Hashing:** รหัสผ่านผู้ใช้ทั้งหมดถูกเข้ารหัสด้วยอัลกอริทึม `bcrypt` ที่มี Salting หลายรอบ ทีมงานไม่สามารถอ่านรหัสผ่านของคุณได้
> * **XSS & CSRF Prevention:** Next.js และ React จัดการกรองและทำความสะอาด Input ทั้งหมดเพื่อความปลอดภัยของข้อมูล
> * **Artwork Protection:** ไฟล์ผลงานต้นฉบับความละเอียดสูงของ Creator จะถูกเก็บไว้ในโฟลเดอร์ปิดบน Storage โดยผู้ใช้ทั่วไปจะสามารถเข้าถึงได้เฉพาะภาพ Preview ขนาดเล็กที่มีการแปะลายน้ำเพื่อป้องกันการถูกละเมิดลิขสิทธิ์

## 2. คู่มือสำหรับลูกค้า (Buyer's Guide)

คู่มือฉบับนี้รวบรวมขั้นตอนและรายละเอียดการสั่งซื้อสำหรับลูกค้า เพื่อให้คุณได้รับเสื้อสกรีนที่มีคุณภาพและดีที่สุด

### 2.1 วิธีการสั่งซื้อสินค้า
1. **เลือกรูปแบบเสื้อ:** เลือกทรงเสื้อ สีเสื้อ และไซส์ที่ต้องการ
2. **อัปโหลดภาพ/เลือกลาย:** 
   * หากต้องการออกแบบเอง ให้กดปุ่ม **Custom** เพื่อเข้าสู่เครื่องมืออัปโหลดภาพ (รองรับ JPG, PNG)
   * หากต้องการซื้อดีไซน์สำเร็จรูป สามารถเลือกแบบจาก Marketplace ในหน้าแรกได้เลย
3. **ปรับแต่งและตรวจดูตัวอย่าง:** ย่อ/ขยาย หมุนภาพ หรือจัดตำแหน่งสกรีนภายในขอบเขตกรอบสกรีนของตัวเสื้อจำลอง
4. **เพิ่มลงตะกร้าและจ่ายเงิน:** ตรวจสอบความถูกต้องและดำเนินการชำระเงินผ่านช่องทางที่สะดวก

### 2.2 การเตรียมไฟล์ภาพสำหรับสกรีน
เพื่อให้ได้ลายสกรีนที่คมกริบ ไม่แตก และสีสันชัดเจน โปรดทำตามคำแนะนำต่อไปนี้:
* **ประเภทไฟล์:** แนะนำเป็น **.PNG ที่ตัดพื้นหลังใส (Transparent background)** หากใช้ไฟล์ .JPG จะมีขอบกรอบสี่เหลี่ยมติดสกรีนไปด้วย
* **ความละเอียด (Resolution):** ควรกำหนดขนาดขั้นต่ำ **2000x2000 พิกเซล** หรือมีความละเอียดอย่างน้อย **300 DPI**
* **โหมดสี (Color Mode):** แนะนำให้ตั้งเป็นโหมดสี **CMYK** หรือ **sRGB**
* **เส้นขอบและตัวอักษร:** ความหนาของเส้นหรือตัวอักษรไม่ควรต่ำกว่า **2pt** เพื่อให้พิมพ์ติดเนื้อผ้าได้ดีที่สุด

### 2.3 ตารางไซส์เสื้อมาตรฐาน (Size Guide)
เสื้อยืดของเราเป็นแบบ **Unisex** ทรงมาตรฐานใส่ได้ทั้งชายและหญิง ตารางขนาดมีดังนี้:

### 2.4 การชำระเงินและบริการจัดส่ง
* **ช่องทางจ่ายเงิน:** สแกนจ่ายด้วย QR Code PromptPay, บัตรเครดิต/เดบิต, และมีบริการเก็บเงินปลายทาง (COD) สำหรับคำสั่งซื้อที่ไม่เกิน 3,000 บาท
* **ระยะเวลารอสินค้า (Make to Order):**
  * **ระยะเวลาผลิต:** ใช้เวลาเตรียมสินค้าและสกรีนลายประมาณ 2-4 วันทำการหลังจากได้รับเงิน
  * **ระยะเวลาจัดส่ง:** ใช้ขนส่งเอกชนชั้นนำ (Kerry / Flash / J&T) ใช้เวลา 1-3 วันทำการขึ้นอยู่กับจังหวัดปลายทาง

### 2.5 นโยบายการเปลี่ยนและคืนสินค้า
* **กรณีรับประกันเคลมสินค้าฟรี 100%:** เสื้อมีตำหนิ ขาด รุ่ย, ลายสกรีนเบี้ยว/พิมพ์ผิดลาย, จัดส่งผิดไซส์ หรือซักแล้วลายลอกในการซักครั้งแรก (แจ้งภายใน 7 วันหลังได้รับพัสดุ โดยที่สินค้าต้องยังไม่ผ่านการซักหรือใช้งาน)
* **กรณีไม่รับคืนหรือเปลี่ยนสินค้า:** ลูกค้าสั่งไซส์ผิด สั่งสีผิด หรือเปลี่ยนใจกะทันหัน เนื่องจากเป็นสินค้าพิมพ์สั่งผลิตใหม่ชิ้นต่อชิ้นเป็นรายบุคคล

## 3. คู่มือสำหรับนักออกแบบ (Creator's Guide)

ช่องทางการสร้างรายได้เสริมจากลายการ์ตูนและภาพวาดของคุณด้วยระบบ Markup Profit แบบกำหนดได้อิสระ

### 3.1 ขั้นตอนการเป็น Creator
1. **ลงทะเบียนบัญชี:** สมัครสมาชิกหรือเข้าสู่ระบบบนแพลตฟอร์ม The Shirtsy
2. **ยื่นสมัคร Creator:** ไปที่โปรไฟล์ เลือกปุ่ม "สมัครเป็น Creator" และกรอกข้อมูลยืนยันตัวตน
3. **ตกแต่งร้านค้า:** ตั้งชื่อร้านค้า อัปโหลดรูปโปรไฟล์ และภาพปกหน้าร้านค้าของคุณ
4. **ผูกบัญชีธนาคาร:** ตั้งค่าการโอนรายได้เพื่อรองรับการจ่ายเงินส่วนแบ่งเมื่อลูกค้าสั่งซื้อผลงาน
5. **อัปโหลดผลงานชิ้นแรก:** เมื่อบัญชีผ่านการอนุมัติ สามารถเริ่มลงขายผลงานได้ทันที!

### 3.2 ข้อตกลงการใช้งานและการละเมิดลิขสิทธิ์
> [!WARNING]
> แพลตฟอร์มมีกฎเหล็ก **"Zero Tolerance"** ต่อการลอกผลงาน
> * ผลงานอาร์ตเวิร์คต้องเป็นของคุณเอง หรือมีใบอนุญาตในการจำหน่ายเชิงพาณิชย์
> * ห้ามนำลายที่เป็นของแบรนด์ดัง ภาพยนตร์ ศิลปิน ดารานักแสดง หรือตัวการ์ตูนลิขสิทธิ์ (รวมถึง Fan Art เชิงพาณิชย์) มาลงขาย
> * หากได้รับการแจ้งเตือนเรื่องการละเมิดลิขสิทธิ์ ผลงานจะถูกนำออกทันที และหากกระทำผิดซ้ำ บัญชี Creator จะถูกแบนถาวรและระงับการโอนส่วนแบ่งรายได้ทั้งหมด

### 3.3 ส่วนแบ่งรายได้และรอบจ่ายเงิน (Payouts)
* **การตั้งราคา:** ใช้สูตรคำนวณ: `[ราคาต้นทุนเสื้อปล่า + ค่าบริการสกรีน] + [กำไร/Markup ที่คุณกำหนดเอง] = [ราคาขายจริงหน้าร้าน]`
* **ตัวอย่าง:** หากต้นทุนผลิตเสื้อคือ 190 บาท และคุณกำหนด Markup ไว้ที่ 100 บาท ราคาขายหน้าร้านจะแสดงเป็น 290 บาท เมื่อลูกค้าทำการสั่งซื้อและชำระเงินสำเร็จ คุณจะได้รับกำไร 100 บาททันที (หักค่าธรรมเนียมธุรกรรม 3-5% ของสถาบันการเงินปลายทาง)
* **รอบโอนเงิน:** ระบบจะสรุปรายได้สะสมของ Creator ทุกๆ สัปดาห์ และโอนเงินเข้าบัญชีที่ตั้งค่าไว้แบบอัตโนมัติภายในวันทำการถัดไป

## 4. คำถามที่พบบ่อย (FAQ)

### ถาม: เสื้อสกรีนสามารถซักเครื่องได้ปกติหรือไม่?
**ตอบ:** สามารถซักเครื่องได้ปกติครับ แต่เพื่อรักษาสภาพลายสกรีนให้อยู่นานที่สุด แนะนำให้กลับเสื้อเอาด้านลายเข้าข้างในก่อนนำไปซัก หลีกเลี่ยงการใช้น้ำยาซักผ้าขาวหรืออบด้วยความร้อนจัด และห้ามรีดทับลายสกรีนโดยตรง

### ถาม: หากต้องการสั่งทำเสื้อจำนวนมากเพื่อแจกงานแต่ง หรือเป็นเสื้อห้อง จะได้ส่วนลดพิเศษไหม?
**ตอบ:** มีแน่นอนครับ หากสั่งผลิตตั้งแต่ 20 ตัวขึ้นไป จะเข้าสู่เรทราคาส่ง กรุณาติดต่อทีมงานซัพพอร์ตโดยตรงเพื่อขอใบเสนอราคาพิเศษ

### ถาม: สามารถส่งภาพถ่ายครอบครัวเพื่อสกรีนบนเสื้อได้หรือไม่?
**ตอบ:** ทำได้เลยครับ โดยการไปที่ปุ่ม "Custom" และทำการอัปโหลดไฟล์ภาพถ่ายคุณภาพสูงของคุณลงไป และทำการจัดวางตำแหน่งตามต้องการ
